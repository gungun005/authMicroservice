# Auth Microservice

The Auth Microservice handles user authentication including login, registration.

## Features

- User registration
- User login
- Secure password hashing with bcrypt

## Tech Stack

- **Node.js** / **Express.js**
- **MongoDB** (optional, for storing user data)
- **bcrypt** (for password hashing)
